---
description: Start the Escola de IA Development Environment
---
To run the full stack application locally:

1.  **Start the Backend**:
    *   Open a new terminal.
    *   Navigate to the backend directory:
        ```bash
        cd backend
        ```
    *   Install dependencies (if not done):
        ```bash
        npm install
        ```
    *   Start the server:
        ```bash
        npm start
        ```
    *   The server will run on `http://localhost:3000` and connect to MongoDB Atlas.

2.  **Start the Frontend**:
    *   Open a second terminal.
    *   Navigate to the frontend directory:
        ```bash
        cd frontend
        ```
    *   Install dependencies (if not done):
        ```bash
        npm install
        ```
    *   Start the development server:
        ```bash
        npm run dev
        ```
    *   The application will be accessible at `http://localhost:5173`.
    *   Access the Admin Dashboard at `http://localhost:5173/admin`.
